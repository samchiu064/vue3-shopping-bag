# 專案名稱：Shopping Bag

一個簡單的 Composition API + Vuex + Tailwind CSS 的購物車練習

## 作品連結

- GitHub Repo: https://github.com/samchiu064/vue3-shopping-bag
- GitHub Page: https://samchiu064.github.io/vue3-shopping-bag/

## 技術要求

- 使用 Vue 3 Composition API
- 使用 Vuex 狀態管理工具
- 使用 Vue Router

## 開發資訊

API 網址：https://fakestoreapi.com/

## 專案需求

- 網站需要有商品列表、購物車兩個頁面
- 商品列表頁需要列出產品清單
- 使用者可以在商品列表頁將商品加入購物車
- 使用者可以在商品列表頁將商品從購物車移除
- 購物車頁面需要列出已經加入購物車的商品資訊
- 使用者可以在購物車頁面將商品加入購物車
- 使用者可以在購物車頁面將商品從購物車中移除
- 使用者可以在購物車頁面調整商品數量
- 當購物車頁面沒有商品時，需顯示目前沒有商品加入購物車提示
